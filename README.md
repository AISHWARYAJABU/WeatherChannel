## ▶️ How to Run

1. Clone the repository:
   git clone https://github.com/AISHWARYAJABU/WeatherChannel.git

2. Open the project in IDE (Eclipse / IntelliJ)

3. Install dependencies:
   - Make sure Maven is installed
   - Update project (Maven -> Update Project)

4. Configure WebDriver:
   - Download ChromeDriver
   - Set path in your system or in code

5. Run test cases:
   - Right-click on TestNG file / test class
   - Click "Run as TestNG Test"
   OR
   - Run command: mvn test
  
## 🔌 API Setup

- Ensure internet connection is active
- If API key is required, add it in config file
- Validate API response using test cases
